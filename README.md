# NewsPulse – Real-time News Web Application

A modern and responsive news web application built using **HTML, CSS, and JavaScript**.
It fetches real-time news using the **News API** and displays them in a clean, interactive UI.

---

##  Features

*  Live news fetching using News API
*  Category-based filtering (Cricket, India, Technology, Politics)
*  Search functionality for custom news queries
*  Infinite scroll for continuous news loading
*  Dark mode toggle
*  Bookmark system using localStorage
*  Fully responsive design

---

##  Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* News API

---

##  How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser

---

##  API Used

News API: https://newsapi.org/
